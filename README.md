Итоговый проект «ValutaTrade Hub» — консольное приложение на Python для управления виртуальным портфелем фиатных и криптовалют с возможностью получения актуальных курсов.

Приложение позволяет регистрировать пользователей и вести мультивалютные кошельки, выполнять операции покупки и продажи валют (BTC, ETH, SOL, EUR, RUB, GBP) за базовую валюту USD, получать текущие курсы через внешние API, рассчитывать общую стоимость портфеля в выбранной валюте, а также сохранять историю курсов и логировать все операции в файл.

Проект имеет модульную структуру и включает логику работы с пользователями и активами, сервис получения валютных котировок, механизм логирования и хранение данных в формате JSON. Точка входа в приложение расположена в файле main.py, конфигурация проекта и зависимостей описана в pyproject.toml, для автоматизации используются команды Makefile.

Запуск проекта осуществляется через Poetry после установки зависимостей. Приложение предназначено для демонстрации навыков работы с Python, структурирования кода, обработки данных и использования внешних сервисов.
